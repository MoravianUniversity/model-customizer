# model-customizer
Customize 3D Models

## How to Setup Openscad CLI (Mac)
- Linux
  - run `sudo install.sh`
- Mac
  - run install.sh
  - Pre-Ventura (macOS 13 or before)
    - go to System Preferences -> Security and Privacy
    - click `Open anyway`
  - Post-Ventura (macOS 14+)
    - go to `/Applications`
    - right click `OpenSCAD.app` and click open
